# Contributing to mssql-tds

Thank you for your interest in this project!

## Current Status

This repository is **not currently accepting external pull requests**. The codebase is under active development by the Microsoft SQL Server drivers team and is published as open source for transparency and community engagement.

## How You Can Help

While we are not accepting code contributions at this time, we welcome:

- **Bug reports** — File an issue using the [bug report template](https://github.com/microsoft/mssql-rs/issues/new?template=bug_report.yml).
- **Feature requests** — Suggest improvements using the [feature request template](https://github.com/microsoft/mssql-rs/issues/new?template=feature_request.yml).
- **Feedback** — Share your experience using the library in GitHub Issues.

## Security

Please report security vulnerabilities through the Microsoft Security Response Center (MSRC), **not** through GitHub Issues. See [SECURITY.md](SECURITY.md) for details.

## Code of Conduct

This project has adopted the [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/). See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) for details.
