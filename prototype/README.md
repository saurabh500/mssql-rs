# Prototype projects

| Project | Description |
|---------|-------------|
| [connect-async](connect-async/connect-async-lib/README.md) | Connects to the SQL server by sending a TDS prelogin and login packets. The implementation uses asynchronous I/O operations. |
| [connect-sync](connect-sync/connect-sync-lib/README.md) | Connects to the SQL server by sending a TDS prelogin and login packets. The implementation uses synchronous I/O operations. |
| [ffi-protype](ffi-prototype/README.md) | Describes potential methods for transferring data between a driver code and the Rust TDS library. |
| [transport](transport/README.md) | The prototype that connects, runs a SQL query and parses result. |
